## Getting Started

Make sure you have Node.js installed.

#### Clone

```bash
git clone https://github.com/enes-erenn/ChitChat.git
```

#### Install

```bash
cd ChitChat
```

```bash
npm install
```

#### Start

```bash
npm run dev
```

## Dependencies

- React
- React-router-dom
- Next
- Sass
- Uuid
- Firebase
- Typescript

## Features

- Firebase Authentication (Register / Login with email)
- Searchs for contacts
- Sending Messages (Text or Image)
